Shooting Stars

Description:

This is a Javascript-based generative web application. Please use the controls in the top-right corner to manipulate the shooting stars and the sky to your desire!

Parameters:

  • Speed: Changes the speed of the shooting stars.
  • Tail_Length: Changes the length of the shooting star's trail.
  • Tail_Width: Changes the width of the shooting star's trail.
  • Radius_1: Changes the radius of the star's indents.
  • Radius_2: Changes the radius of the star's points.
  • Points: Changes the number of points the star has.
  • Description_Light/Dark: Change the text from light theme to dark theme.
  • Stars: Changes the color of the stars.
  • Tail_Color: Changes the color of the shooting star's trail.
  • Background: Changes the color of the sky.
